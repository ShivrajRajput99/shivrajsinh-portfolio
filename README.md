# Shivrajsinh Rajput Portfolio

Modern personal portfolio website built with React, Vite, and Tailwind CSS.

## Features

- Dark glassmorphism UI
- Responsive hero, about, skills, projects, timeline, and contact sections
- Reusable React components
- SEO metadata in `index.html`
- Accessible labels, focus states, and reduced-motion support
- Sample project and profile data in `src/data/portfolio.js`

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Update the contact details, project URLs, and resume link in `src/data/portfolio.js`.
